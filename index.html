Formula 1 - Exploratory Data Analysis


This project presents an Exploratory Data Analysis (EDA) of the Formula 1 2023 season race results. It offers insights into driver and team performances, race outcomes, and key statistics. Key analyses include identifying top drivers by track, average driver positions, team podium finishes, and the impact of starting grid positions. This comprehensive analysis aids in evaluating performance, strategic planning, and provides valuable insights for fans and analysts. The project showcases proficiency in aggregations, window functions, and data visualization.

Mortgage Amortization Calculator


This repository features an Excel-based Mortgage Calculator designed to compute monthly mortgage payments, total payments, and total interest for a given loan. It includes a comprehensive amortization table, breaking down each payment into principal and interest components. The calculator utilizes the PMT formula, along with Excel functions like SUM and cell referencing, to dynamically update calculations based on user inputs. Additional features include property taxes, HOA fees, and property insurance, providing a holistic view of monthly housing costs, aiding in better financial planning and decision-making.
